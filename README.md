In the world of open source, **consistency is king.** But while our code evolves every day, our GitHub Profile READMEs often stay frozen in time. We write "Active Developer" in our bios, yet the timestamps tell a different story.

**Calendar-by-Cygra** was built to bridge that gap. This project provides a lightweight, high-performance API that serves a dynamic image file reflecting the exact date of the current day. No more manual edits, no more outdated info‚Äîjust a clean, automated badge that proves you are here, active, and building in real-time. ‚ö°


### üõ†Ô∏è What it Does
This API generates a live-updating **"Daily Pulse"** badge for your profile. By serving a direct image file based on real-time data, it ensures that every visitor sees the most current information the moment they land on your page.

* **Real-time Accuracy:** Syncs with global time standards.
* **Zero Maintenance:** Set it once, and let the API handle the rest.
* **Minimalist Design:** Fits perfectly into any GitHub aesthetic.

## ‚öôÔ∏è API Parameters

Customize your calendar widget by passing these query parameters in the URL.

| Parameter | Type | Description | Default |
| :--- | :---: | :--- | :--- |
| `width` | `number` | The total width of the generated image (px). | `1200` |
| `height` | `number` | The total height of the generated image (px). | `100` |
| `textColor` | `string` | Hex code for text. Use `%23` for `#` (e.g., `%23ffffff`). | `#000000` |
| `bgColor` | `string` | Hex code for background. Supports `transparent`. | `#ffffff` |
| `fontSize` | `number` | Size of the text in pixels. | `60` |
| `font` | `string` | Choose a font family. See [Available Fonts](#fonts). | `cheese-orange` |
| `radius` | `number` | Corner roundness. Use `50` for a pill shape. | `0` |
| `title` | `string` | The custom message to display on the image. | `Welcome!` |
